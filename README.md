## Gym Membership System

This is a Simple Gym membership system used for checking in the members and payments.

## Functions -
1. Add Member
2. View All Members
3. Make Payment
4. View Payments
5. Check-In
6. Check-Out
7. View Entry&Exit Tables
8. View Plans
9. Exit
